# EduManager — Système de Gestion des Étudiants

Application web dynamique de gestion des étudiants développée dans le cadre du module Java EE.

## Stack Technique
- **Backend** : Java EE (Servlets + JSP)
- **Architecture** : MVC + DAO
- **Base de données** : MySQL
- **Serveur** : Apache Tomcat
- **Gestion de projet** : Maven
- **Frontend** : HTML + CSS

## Fonctionnalités
- Authentification administrateur avec session
- Ajouter un étudiant
- Modifier les informations d'un étudiant
- Supprimer un étudiant
- Afficher la liste des étudiants
- Rechercher un étudiant par nom ou CNE

## Installation
1. Importer le projet dans Eclipse IDE
2. Configurer MySQL et créer la base de données
3. Modifier les paramètres de connexion dans `db.properties`
4. Déployer sur Apache Tomcat
5. Accéder sur : `http://localhost:8080/gestion-etudiants`
